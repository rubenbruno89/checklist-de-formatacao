# 🛠️ Checklist de Formatação Profissional

Um checklist interativo, editável e responsivo para técnicos de TI organizarem o processo de formatação e manutenção de computadores. Desenvolvido para ser simples, eficiente e fácil de compartilhar com a equipe via Microsoft Teams.

🚀 **Acesse a versão online aqui:** [https://rubenbruno89.github.io/checklist-de-formatacao/](https://rubenbruno89.github.io/checklist-de-formatacao/)

![Preview do Checklist](https://via.placeholder.com/800x400?text=Checklist+de+Formatacao+Preview) *(Substitua por um screenshot real se desejar)*

## ✨ Funcionalidades

- ✅ **Totalmente Editável**: Adicione, remova ou edite itens em qualquer categoria diretamente na interface.
- 💾 **Salvamento Automático**: Seus dados e progressos são salvos localmente no navegador (LocalStorage). Não perca seu trabalho!
- 📋 **Integração com Teams**: Botão dedicado para copiar o checklist formatado com emojis e estrutura pronta para colar no Microsoft Teams.
- 📊 **Barra de Progresso**: Visualize em tempo real quanto do serviço já foi concluído.
- 🖨️ **Impressão/PDF**: Gere um relatório limpo em PDF para entregar ao cliente ou arquivar.
- 📱 **Responsivo**: Funciona perfeitamente em desktops, tablets e celulares.
- 🌐 **Offline First**: Funciona sem internet após o primeiro carregamento.

- Técnicos de manutenção de computadores.

## 🚀 Como Usar

1. **Acesse o Site**: Clique no link [https://rubenbruno89.github.io/checklist-de-formatacao/](https://rubenbruno89.github.io/checklist-de-formatacao/)
2. **Preencha os Dados**: Insira nome do cliente, telefone e tipo de equipamento.
3. **Execute o Serviço**: Vá marcando os checkboxes conforme realiza cada etapa.
4. **Personalize (Opcional)**:
   - Clique no texto de qualquer item para **editar**.
   - Clique em **"➕ Adicionar novo item"** para incluir etapas específicas.
   - Clique no ícone 🗑️ para remover itens desnecessários.
5. **Compartilhe**:
   - Clique em **"📋 Copiar para Teams"** e cole na conversa do seu time.
   - Ou clique em **"🖨️ Imprimir / PDF"** para gerar um documento físico/digital.

## 📂 Categorias do Checklist

O checklist vem pré-configurado com 8 etapas essenciais:

1. 💾 **Backup de Dados**: Garantir que nada seja perdido.
2. 🔧 **Preparação**: Drivers, licenças e limpeza física.
3. ⚙️ **Formatação**: Instalação do Windows e configurações básicas.
4. 🔄 **Drivers e Atualizações**: Deixar o sistema atualizado e estável.
5. 📦 **Softwares Essenciais**: Navegadores, Office, antivírus, etc.
6. ⚡ **Configurações do Sistema**: Otimização de energia, privacidade e inicialização.
7. 📂 **Restauração de Dados**: Retornar arquivos do cliente para o novo sistema.
8. ✅ **Testes Finais**: Verificar hardware, internet e entregar ao cliente.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica.
